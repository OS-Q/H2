[P14](https://github.com/OS-Q/P14) 用于 [STM8S](https://www.st.com/zh/microcontrollers-microprocessors/stm8s-series.html) 开发

#### 基本特性

* 直接IO控制
* UART/IIC通信
* EEPROM数据存储
